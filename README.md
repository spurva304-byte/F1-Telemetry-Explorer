# F1 Telemetry Explorer 🏎️📊

Analyze and visualize Python Formula 1 telemetry data.  
The project will give you the opportunity to compare the performance of various drivers through the analysis of the fastest laps and speed traces of the various drivers in a race track.

---

## 🚀 Features
- Load actual F1 session information with FastF1.  
- Extract and analyze fastest lap times.  
- Compare multiple drivers  
- Plot graphs of speed vs distance.  
- Adaptable input (any driver, any race)

---

## 🧠 What You’ll Learn
- Using real-life data in sports.  
- Data analysis using Python  
- Formula 1 concepts of telemetry.  
- Bar chart with Matplotlib.  

---

## 🛠 Tech Stack
- Python  
- FastF1  
- Matplotlib  

---

## 📊 Output
Produces a speed comparison graph of the performance of the various drivers along the track.
